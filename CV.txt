ПРОЕКТЫ (https://github.com/MatveiV)

- ShortLongMemory Bots (https://github.com/MatveiV/ShortLongMemoryBots): серия Telegram-ботов с короткой (история диалога, deque) и долгой (RAG + ChromaDB) памятью. Универсальный бот с выбором режима, FSM-конфигуратором, интеграцией трёх AI-провайдеров и поддержкой 16+ моделей. Документация с 8 диаграммами.

- Local AI Agent Bot (https://github.com/MatveiV/Local_AI_agent_bot): терминальный и Telegram AI-агент с OpenAI function calling и 11 встроенными инструментами. Два режима интерфейса (CLI и Telegram-бот) с единым ядром агента. Поддержка трёх AI-провайдеров (Z.AI, ProxyAPI, GenAPI) и 13+ моделей (GPT-4.x, GLM-4.x, Claude, Gemini, DeepSeek). Инструменты: поиск DuckDuckGo, погода, курсы крипты и валют, генерация QR-кодов, расписание/напоминания, калькулятор, чтение PDF/DOCX/TXT, файловый ввод-вывод, HTTP-запросы, терминальные команды. Документация с 5 Mermaid-диаграммами.

- LangChain Pipeline Generator: инструмент для автоматической генерации Telegram-ботов и документации (SRS, URS) по текстовому описанию с помощью LLM-цепочек.

- Prompter: CLI-инструмент для A/B-тестирования техник промптинга (zero-shot, few-shot, chain-of-thought, role-based) с автоматическим ранжированием ответов и генерацией Markdown/DOCX-отчётов. Интеграция с тремя AI-провайдерами.

- PromptingAIbot & three_ai_comparison_bot — Telegram-бот и набор CLI-скриптов для сравнения ответов AI-моделей (GLM, GPT, Claude, Gemini, DeepSeek) через единый OpenAI-совместимый клиент. Поддержка контекста диалога, управление сессиями, отчёт о стоимости запросов в рублях.

- CoinParser: парсер Telegram-канала для фильтрации списка криптовалютных символов. Интеграция с Google Sheets API, экспорт статистики в XLSX.

- Loki-Grafana: стек централизованного логирования (Loki + Grafana) с эмулятором крипто-бэкенда, развёртыванием через Docker Compose и автоматическими bash-скриптами установки.

- ML_Fin_Notebooks: ноутбуки по машинному обучению в финансах: бэктестинг торговых стратегий, генерация признаков (EMA, RSI, MACD), обучение моделей классификации, прогнозирование временных рядов.

- Еще проекты: несколько чат-ботов с ИИ в телеграмм; бот для сравнения цен криптовалют на разных биржах в реальном времени; система автоматизированной генерации счетов-фактур с логотипами и доверенностями; приложение для хранения и управления паролями с использованием шифрования; парсер сообщений; приложение для управления заметками; win-приложение для напоминаний задач c интерфейсом на Tkinter и системными уведомлениями; расписание для детского спортивного клуба, БД фильмов и др.

ДОСТИЖЕНИЯ:
- Реализовал единый OpenAI-совместимый клиент, работающий с тремя разными провайдерами (Z.AI, ProxyAPI, GenAPI) без изменения кода.
- Реализовал AI-агента с 11 инструментами на базе OpenAI function calling: агент самостоятельно выбирает инструмент по запросу на естественном языке, выполняет цепочку вызовов (до 5 итераций) и формирует структурированный ответ. Архитектура с callback-интерфейсом позволила подключить два независимых UI (CLI и Telegram) к единому ядру без дублирования кода.
- Реализовал универсальный Telegram-бот с тремя режимами памяти (короткая, долгая, комбинированная) в одном скрипте; документы сохраняются между перезапусками (ChromaDB persistent), данные пользователей изолированы.
- Реализовал три независимые 4-звенные цепочки (бот, SRS, URS) с автоматическим ревью и исправлением результата; сгенерированный бот запускается без ручных правок; документы содержат встроенные архитектурные диаграммы (C4 Context, C4 Container, UML Use Case, Sequence, Component).
- Провёл сравнительное тестирование 6 AI-моделей на 3 системных промптах, оформил результаты в структурированный DOCX-отчёт с выводами и рекомендациями по оптимальным параметрам.
- Разработал инструмент автоматической оценки качества промптов с формульным ранжированием (json_valid, steps_count, avg_notes_len).
- Построил и задокументировал стек логирования Loki + Grafana с нуля, включая bash-скрипты автоматического развёртывания на Ubuntu.
- Разработал и протестировал набор алгоритмических торговых стратегий (https://github.com/MatveiV/ML_Fin_Notebooks), показавших на исторических данных стабильный результат, подтвердив жизнеспособность использования методов ML в финансовом анализе. (Модели ML для прогнозирования финансовых временных рядов: scikit-learn, pytorch, модели LSTM и TFT).
