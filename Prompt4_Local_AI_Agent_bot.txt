# TASK
Создай локального AI-Агента, работающего через OpenAI, который можно запускать из терминала.
Агент должен уметь:
- понимать задачи на естественном языке
- выбирать подходящее действие
- выполнять внешние инструменты

# AGENT CAPABILITIES (инструменты):
1) Web Search (DuckDuckGo)
2) HTTP API Requests (через requests)
3) Доступ к файловой системе (чтение/запись)
4) Выполнение терминальных команд (ограниченно, безопасно)
5) Работа с контекстной памятью (сохранение истории диалога в файл)

#Initial setup
Скрипт локального AI-Агента должен подключать openai или любой совместимый LLM (gpt-4, gpt-3.5, llama 3 и т.д.), который получен через AI-провайдера, так же как это сделано в скриптах ai_direct.py и openai_client.py с теми же api-ключами как и в файле .env.
OPENAI_API_KEY не нужен отдельно, а от того провайдера, что выбрал пользователь.

# BOT LOGIC
0. Пользователь выбирает ai-провайдера, модель, температуру, токены из доступных вариантов в скриптах ai_direct.py и openai_client.py.

# ДОПОЛНИТЕЛЬНЫЕ ИНСТРУМЕНТЫ (ЭТО ВАЖНО):

## Инструмент: Получить погоду
Используй бесплатный API:
https://api.open-meteo.com/v1/forecast
Запрос:
GET https://api.open-meteo.com/v1/forecast?latitude={lat}&longitude={lon}¤t_weather=true
Возвращаемое значение: температура, ветер, условия.
Функция должна принимать название города → преобразовывать в координаты (через geocoding), затем получать погоду.

## Инструмент: Узнать курс криптовалюты
Используй CoinGecko API (без ключей):
GET https://api.coingecko.com/api/v3/simple/price?ids={coin}&vs_currencies={currency}
Напр: bitcoin, ethereum
currency: usd, eur, rub и т.д.
Функция: get_crypto_price(coin: str, currency: str) → float.

# PROGRAM STRUCTURE
Создай проект со следующей структурой:
agent/
  ├── agent.py          # логика агента
  ├── tools.py          # подключение инструментов (включая погоду и крипту)
  ├── memory.json       # долговременная память агента
  ├── run.py            # CLI-интерфейс запуска агента
  └── .env              # хранит OPENAI_API_KEY

# BEHAVIOR LOGIC
При запросе пользователя:
1) Агент анализирует задачу.
2) Выбирает инструмент:
    - Если нужна информация → Web Search
    - Если нужна погода → Weather Tool
    - Если нужна цена крипты → Crypto Price Tool
    - Если действие → Terminal Exec
    - Если данные → File IO
3) При недостатке контекста → спрашивает уточнение.
4) Добавляет резюме диалога в memory.json
5) Возвращает понятный и структурированный ответ.

# OUTPUT FORMAT
Верни:
1) Полный Python код всех файлов
2) Инструкции по запуску:
   pip install -r requirements.txt
   python run.py
